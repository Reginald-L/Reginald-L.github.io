FashionMNIST

概述

FashionMNIST数据集包含了10种类别的共7万个商品的图片. 其中6万个用作训练集, 1万个用作测试集. 每一张图片都是 $ 12828 $ 的灰度图片. 官方描述如下:

Fashion-MNIST is a dataset of Zalando’s article images—consisting of a training set of 60,000 examples and a test set of 10,000 examples. Each example is a 28x28 grayscale image, associated with a label from 10 classes. We intend Fashion-MNIST to serve as a direct drop-in replacement for the original MNIST dataset for benchmarking machine learning algorithms. It shares the same image size and structure of training and testing splits. (https://github.com/zalandoresearch/fashion-mnist)

数据集下载

训练样本一共60000条, 测试数据一共10000条. 每一个样本都是一个长度为2的tuple. 第一个元素是一个三维tensor类型, 即 $(12828)$, 一个值表示一个像素; 第二个元素是标签, 比如9.
